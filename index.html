Buzzr is a hands-on electronics tutorial.

Buzzr takes a different approach to teaching electronics. We don't believe in studying electronics on books; rather, we think electronics is best taught when you can actually see how things work.

Download

Check out the latest release for binaries, or clone the project and build from source: 


$ git clone https://github.com/CapacitorSet/buzzr
$ cd buzzr/src
$ make jar
Alternatively, if you don't have make, you can build using javac *.java.
